# **App Name**: Kickoff Kasse

## Core Features:

- Nutzerauthentifizierung & Rollenverwaltung: Spieler können sich registrieren und anmelden. Das System unterscheidet zwischen regulären Spielern und Kassenprüfern mit erweiterten Rechten, wie dem Bearbeiten aller Kosten oder dem Hinzufügen neuer Spieler.
- Spielerverwaltung: Kassenprüfer können neue Spielerprofile anlegen, um sie in die Abrechnung einzubeziehen. Spieler können ihre eigenen Profile einsehen und grundlegende Informationen bearbeiten.
- Getränkekosten erfassen (Spieler): Reguläre Spieler können ihren eigenen Konsum von Bieren (1,50€ pro Einheit) oder ganzen Kisten (35€ pro Kiste) eintragen, der ihnen dann zugeordnet wird.
- Getränkekosten erfassen (Kassenprüfer): Kassenprüfer haben die Berechtigung, Getränkekosten für jeden beliebigen Spieler oder direkt für die 'Mannschaftskasse' einzutragen und zu verwalten.
- Gesamtkostenübersicht & Historie: Alle Spieler können eine detaillierte Übersicht über offene Kosten einsehen, die nach Datum sortiert und wahlweise nach Spieler gefiltert werden kann. Der aktuelle Stand der Mannschaftskasse ist ebenfalls ersichtlich.
- AI-gestütztes Kassenprüfer-Tool: Ein generatives AI-Tool, das den Kassenprüfern hilft, automatische Zusammenfassungen der Teamausgaben zu erstellen, überfällige Zahlungen hervorzuheben und mögliche Zahlungsaufforderungen zu formulieren.

## Style Guidelines:

- Primary color: A desaturated, professional blue (#3986AB) to convey trust and clarity in financial tracking.
- Background color: A very light, almost white shade of the primary blue (#F2F5F6) for a clean and unobtrusive interface.
- Accent color: A vibrant, bright cyan (#52DDE3) provides an energetic contrast for interactive elements and highlights, suggesting dynamic team spirit.
- All text uses 'Inter' (sans-serif) for a modern, objective, and highly legible experience across all display sizes, suitable for both headlines and body text.
- Utilize a consistent set of clean, line-style icons. These include football-related symbols, currency signs for financial entries, and user avatars to personalize profiles and transactions.
- A mobile-first responsive layout prioritizing quick access to cost entry and a clear, summarized overview of finances on small screens. Important actions should be easily accessible through floating action buttons or prominent navigation.
- Subtle and functional animations will provide visual feedback for user interactions, such as transitions when filtering cost lists, successful submission confirmations, and loading states for data synchronization.
